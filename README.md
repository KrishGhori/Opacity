# Image Opacity Hover Effect

## Project Description

This project demonstrates how to use **CSS opacity and hover effects** on images.
When the user moves the mouse over an image, its transparency changes smoothly using CSS transitions.

## Features

* Two images displayed on the webpage
* Hover effect changes image opacity
* Smooth transition animation
* Simple and clean UI
* Built using HTML and CSS only

## Technologies Used

* HTML5
* CSS3

## How It Works

* The first image is fully visible.
* When the user hovers over it, the opacity becomes **0.3**.
* The second image starts with **low opacity (0.3)**.
* When the user hovers over it, it becomes **fully visible (opacity 1)**.
* CSS `transition` is used to make the effect smooth.
